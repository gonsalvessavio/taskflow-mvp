# Taskflow MVP

🎯 A simple, shareable project management app built for indie makers and solo teams.

## 🔥 MVP Goals
- ✅ Task list with login
- 🔄 Kanban + Calendar views
- 🤝 Referrals to invite friends
- 💳 $5/month subscription via Stripe

## 🧰 Tech Stack
- React + Tailwind (Vite)
- Supabase (Auth + DB)
- Vercel (Hosting)
- Stripe (Payments)
